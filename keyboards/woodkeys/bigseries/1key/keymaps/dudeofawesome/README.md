# DudeOfAwesome's Big Series 1-key layout

## Features

### Tap dancing taps:
1. Media Play / Pause
1. Media Next
1. RGB Mode Next
1. RGB Mode Previous

## Building and flashing

1. Put your board in DFU mode with the button on the bottom
1. Flash:
    ```bash
    $ make bigseries:dudeofawesome:dfu
    ```
