# GMMK3 Pro 75% ANSI Default Layout

This is the default layout for this keyboard.

## Fn layer bindings

This layout has the following bindings for the Fn layer (which is activated by holding the `Fn` key and pressing the corresponding key of the binding):

| Key  | Function |
|------|----------|
| `F1` | Launch My Computer. |
| `F2` | Browser home. |
| `F3` | Launch calculator. |
| `F4` | Launch media player. |
| `F5` | Previous media track. |
| `F6` | Next media track. |
| `F7` | Play/Pause media track. |
| `F8` | Stop media track. |
| `F9` | Mute toggle. |
| `F10` | Volume down. |
| `F11` | Volume up. |
| `F12` | Print screen. |
| `Z` | RGB previous zone. |
| `X` | RGB next zone. |
| `C` | RGB animation speed down. |
| `V` | RGB animation speed up |
| `B` | RGB color saturation down. |
| `N` | RGB color saturation up. |
| `Space` | Toggle RGB. |
| `Up arrow` | RGB brightness up. |
| `Down arrow` | RGB brightness down. |
| `Left arrow` | RGB effect previous. |
| `Right arrow` | RGB effect next. |

In Window layer, the binding `Fn+Win` also toggles the Window key.
