## Function Layer

* `Fn` + `F5`  =  Play
* `Fn` + `F6`  =  Stop
* `Fn` + `F7`  =  Previous Track
* `Fn` + `F8`  =  Next Track
* `Fn` + `F9`  =  Toggle GUI key
* `Fn` + `F10` =  Mute
* `Fn` + `F11` =  Volume Down
* `Fn` + `F12` =  Volume Up
* `Fn` + `Pause` = Reset to Bootloader
