# Default Keymap for TS Southpaw

This is the default keymap for the TS Southpaw keyboard. It includes a base layer with a standard QWERTY layout plus southpaw numpad, and a function layer with RGB lighting controls.

## Features

- Southpaw numpad layout (numpad on the left side)
- Volume control via rotary encoder
- RGB lighting effects and controls
- Arrow keys highlighted in blue when in solid color mode
- ESC key ripple effect (toggle with FN+E)
- Caps Lock and Num Lock indicators
- Microphone mute function with indicator
- Dynamic macros (record/playback with FN+REC1/REC2/PLY1/PLY2)
- Caps Word toggle (FN+CAPS or custom KC_CPSWRD key)

## Layers

### Base Layer
- Standard QWERTY layout
- Left-side numpad
- Media controls
- Function key row (F1-F12)

### Function Layer (Accessed via key to left of right Control)
- RGB lighting controls
- Caps Word toggle
