# Evolv75 PCB QMK firmware

## Introduction

This is the QMK firmware repository for the Evolv75 PCB, a 75% with encoder keyboard designed by NathanAlphaMan in aprtnership with Gondolindrim.

As of november 2025, the Evolv Group Buy has already finished and there is no way to buy a unit anymore. The IC page for the keyboard can be found [here](https://geekhack.org/index.php?topic=104531).

## How to compile

After setting up your build environment, you can compile the Evolv Beta with default keymap by using:

    make evolvbeta:default

See the [build environment setup](https://docs.qmk.fm/#/getting_started_build_tools) and the [make instructions](https://docs.qmk.fm/#/getting_started_make_guide) for more information. Brand new to QMK? Start with our [Complete Newbs Guide](https://docs.qmk.fm/#/newbs).
