# The default keymap for nafuda

## Base
|        1       |     2    |  3    |
|:--------------:|:--------:|:-----:|
|                |   MOUSE  |       |
| ADJUST or BSPC |     UP   |BROWSER|
|    LEFT        |    DOWN  |  RIGHT|


## MOUSE
|  1       |     2    |     3     |
|:--------:|:--------:|:---------:|
|          |    BASE  |           |
|    Btn 1 | MOUSE_UP |    Btn2   |
|MOUSE_LEFT|MOUSE_DOWN|MOUSE_RIGHT|


## BROWSER
|       1     |     2    |    3    |
|:-----------:|:--------:|:-------:|
|             |  LCTL(W) |         |
| LCTL+LSFT(T)| WHEEL_UP | BROWSER |
|LCTL+SFT(TAB)|WHEEL_DOWN|LCTL(TAB)|


## Adjust
|    1    |     2   |    3   |
|:-------:|:-------:|:------:|
|         | RGB_VAD |        |
| Adjust  | RGB_SAD | RGB_VAI|
| RGB_MOD | RGB_TOG | RGB_SAI|
