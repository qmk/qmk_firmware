# Snowkuma's Keymap for [Corne Keyboard (crkbd)](https://github.com/foostan/crkbd)

## Keymap

### COLEMAK
```c
  //,----------------------------------------------------.      ,----------------------------------------------------.
  //      Tab,       Q,       W,       F,       P,      G,              J,       L,       U,       Y,       ;,     \
  //|--------+--------+--------+--------+--------+-------|      |--------+--------+--------+--------+--------+-------|
  //      ESC,       A,       R,       S,       T,      D,              H,       N,       E,       I,       O,     '
  //|--------+--------+--------+--------+--------+-------|      |--------+--------+--------+--------+--------+-------|
  //   LShift,       Z,       X,       C,       V,      B,              K,       M,       ,,       .,       /,     =
  //|--------+--------+--------+--------+--------+-------|      |--------+--------+--------+--------+--------+-------|
                                 //    ï£¿,    Ctrl,  Space,          Enter,  BackSP, Del
                                 //`---------------------'      `---------------------'
```

### LOWER
```c
  //,----------------------------------------------------.      ,----------------------------------------------------.
  //         ,        , KC_BRIU,    Play,    Mute, K_CSCN,           PgUp,    Home,      Up,     End,        ,  K_MDSH
  //|--------+--------+--------+--------+--------+-------|      |--------+--------+--------+--------+--------+-------|
  //         ,        , KC_BRID,    Next,   VolUp,       ,           PgDn,    Left,    Down,   Right,  K_LOCK,
  //|--------+--------+--------+--------+--------+-------|      |--------+--------+--------+--------+--------+-------|
  //         ,        ,        ,    Prev,   VolDn,      [,              ],        ,        ,  M_CODE,  M_XXX1, M_PYNV,
  //|--------+--------+--------+--------+--------+-------|      |--------+--------+--------+--------+--------+-------|
                                 //     ,        ,       ,               ,        ,
                                 //`---------------------'      `---------------------'
```

### RAISE

```c
  //,----------------------------------------------------.      ,----------------------------------------------------.
  //        `,       1,       2,       3,       4,      5,              6,       7,       8,       9,       0,      -
  //|--------+--------+--------+--------+--------+-------|      |--------+--------+--------+--------+--------+-------|
  //        ~,       !,       @,       #,       $,      %,              ^,       &,       *,       (,       ),      _
  //|--------+--------+--------+--------+--------+-------|      |--------+--------+--------+--------+--------+-------|
  //      F11,      F1,      F2,      F3,      F4,     F5,             F6,      F7,      F8,      F9,     F10,    F12

                                 //     ,        ,       ,               ,        ,
                                 //`---------------------'      `---------------------'
```
