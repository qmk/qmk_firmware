# The default keymap for Dozen0

## Base
```
,-----------------------------------------.
|  Cut | Copy |Paste |  Up  |Delete| Bksp |
|      |      |      |      |      |Raise |
|------+------+------+------+------+------|
| Ctrl |Shift | Left | Down |Right |Enter |
|      |      |      |      |      |Lower |
`-----------------------------------------'
```

```
`-----------------------------------------'
```
