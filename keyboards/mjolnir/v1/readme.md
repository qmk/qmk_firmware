# Mjolnir V1

A split keyboard for personal use.

* Keyboard Maintainer: [Martin Liivak](https://github.com/martinliivak)

Compiling the firmware for this keyboard (after setting up your build environment):
```
qmk compile --keyboard mjolnir/v1 --keymap default
```

See the [build environment setup](https://docs.qmk.fm/#/getting_started_build_tools) and the [make instructions](https://docs.qmk.fm/#/getting_started_make_guide) for more information. Brand new to QMK? Start with our [Complete Newbs Guide](https://docs.qmk.fm/#/newbs).

## Bootloader

Enter the bootloader:
* **Physical reset button**: Press the reset button on the back of the MCU for about half a second.
