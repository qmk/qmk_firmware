# IFo Hancroft's ErgoDox EZ Layout

- Layer 0: A QWERTY Layer.
- Layer 1: A Function Layer.
- Layer 2: Empty Layer.
- Layer 3: Empty Layer.

This keymap also does the following:

- Forces NKRO on
- Enables:
    - Via
    - The Swap Hands feature
- Disables:
    - Mouse Key
    - Extra Key
