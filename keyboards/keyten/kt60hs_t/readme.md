# keyten kt60HS-T

60% MX Hot-Swap Tsangan PCB

![kt60HS-T image](https://i.imgur.com/vM32aoX.jpeg)

Supports:
1. Split Backspace
2. Stepped Caps Lock

There are two versions of the PCB available.

|Version|                 Features                  |
|-------|-------------------------------------------|
|v1     |Blue/Purple PCB / ARM STM32F401 controller |
|v2     |Purple PCB / ARM STM32F072 controller      |
